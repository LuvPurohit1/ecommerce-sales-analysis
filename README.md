# E-Commerce Sales Analysis

## Objective
Analyze sales data to identify revenue drivers, profitable segments, and areas of loss.

## Tools Used
- Python (Pandas, NumPy)
- Matplotlib / Seaborn
- Power BI

## Key Insights
- Technology is the most profitable category
- Copiers generate highest profit while Tables cause major losses
- High discounts negatively impact profit
- West region is most profitable, while other regions have higher sales but lower efficiency

## Dashboard
The dashboard highlights:
- Category and sub-category performance
- Regional sales and profit trends
- Impact of discount on profit

## 🚀 Conclusion
This project demonstrates how data analysis can help identify business opportunities and improve profitability.
